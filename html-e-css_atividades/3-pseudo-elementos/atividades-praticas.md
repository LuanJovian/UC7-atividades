### Slide 188:
1. Escreva o código CSS para inserir o ícone "🔍" antes de todos os links de busca em uma página, com um espaçamento de 5px à direita.
2. Crie um código CSS que adicione a palavra "RECOMENDADO" após todos os produtos que possuem a classe produto-destaque, com fundo amarelo e texto em negrito.
3. Desenvolva um CSS que crie um efeito de aspas duplas decorativas (") antes e depois de todas as citações (`<blockquote>`), com tamanho de fonte 3rem e cor cinza.
4. Escreva o código para criar um overlay escuro (fundo preto com 50% de opacidade) sobre imagens com a classe imagem-destaque usando ::before.
5. Crie um CSS que transforme a primeira letra de todos os parágrafos com a classe capitular em uma letra capitular de tamanho 4em, com float à esquerda e margem direita de 8px.
6. Desenvolva um código que destaque a primeira linha de todos os artigos com a classe introducao em negrito e com cor azul escuro.
7. Escreva o CSS para personalizar a cor de seleção de texto de todo o site com fundo laranja ( #ff6600 ) e texto branco.
8. Crie um código que adicione uma seta para baixo "▼" após todos os itens de menu que possuem submenu, com tamanho reduzido e margem à esquerda.
9. Desenvolva um CSS que use ::before para criar uma linha horizontal decorativa (altura 3px, largura 50px, cor vermelha) antes de todos os títulos com a classe titulo-decorado.
10. Escreva um código que utilize ::after para exibir o valor de um data attribute data-status após elementos com a classe status-item.

### Slide 189:

1. Crie um CSS para fazer a primeira letra de cada parágrafo de um conto ter cor verde escuro, fundo amarelo claro e borda arredondada.
2. Desenvolva um código que faça a primeira linha de todos os comentários de usuário terem fonte versalete (small-caps) e cor roxa.
3. Escreva o CSS para que o texto selecionado dentro de áreas de código (`<code>`) tenha fundo preto e texto verde neon (#00ff00).
4. Crie um código que use ::before para criar tooltips simples que aparecem antes de elementos com data-tooltip, mostrando o conteúdo do atributo.
5. Desenvolva um CSS completo que combine ::first-letter e ::first-line para estilizar um parágrafo de abertura, onde a primeira letra seja capitular em vermelho e a primeira linha tenha versalete em azul.

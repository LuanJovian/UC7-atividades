### Slide 238:
1. Crie um container com display: flex e três divs internas coloridas. Aplique justify-content: space-between e observe como os itens se distribuem.
2. Desenvolva uma barra de navegação simples com 5 links lado a lado utilizando Flexbox, sem nenhum espaçamento especial.
3. Crie um container com 3 cards de 200px de largura cada. Utilize Flexbox para que fiquem lado a lado e centralizados horizontalmente na página.
4. Construa um container com 4 itens e aplique flex-direction: column para que fiquem empilhados verticalmente, cada um com margem inferior de 10px.
5. Crie um container com altura de 300px e 3 itens de alturas diferentes. Utilize align-items: center para alinhar todos verticalmente ao centro.
6. Desenvolva uma galeria simples com 6 imagens quadradas de 150px, utilizando flex-wrap: wrap para que quebrem para a linha seguinte quando não couberem.
7. Crie um container com 2 divs: uma com largura fixa de 200px e outra com flex-grow: 1 para ocupar o espaço restante.
8. Construa um menu com 4 itens onde o último item (Sair) seja empurrado para a direita utilizando margin-left: auto.
9. Crie um card simples com uma imagem de 80px à esquerda e um texto à direita, utilizando Flexbox para alinhar corretamente.

### Slide 239:
10. Desenvolva um container com 3 botões lado a lado, centralizados horizontalmente, com um pequeno espaçamento de 10px entre eles utilizando gap.
11. Crie um rodapé com 3 colunas de conteúdo (Sobre, Serviços, Contato) distribuídas igualmente com espaço entre elas.
12. Construa uma lista de tarefas onde cada item tenha um checkbox à esquerda e o texto da tarefa à direita, ocupando toda a largura.
13. Crie um container com 5 tags e utilize flex-wrap: wrap para que as tags que não couberem na linha quebrem para a linha de baixo.
14. Desenvolva um banner simples com um título e um botão, centralizados horizontalmente dentro do banner, com flex-direction: column.
15. Crie uma ficha de contato com ícone, nome e telefone, onde os três elementos fiquem alinhados horizontalmente com espaçamento uniforme.

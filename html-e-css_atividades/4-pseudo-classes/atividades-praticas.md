### Slide 210:
1. Escreva o código CSS para criar um botão que mude a cor de fundo para azul escuro quando o mouse passar sobre ele.
2. Crie um CSS para que todos os links dentro de um menu mudem de cor e ganhem um sublinhado quando estiverem com foco (tecla Tab).
3. Desenvolva um código que faça um card de produto aumentar ligeiramente de tamanho (scale 1.05) e ganhar sombra quando o mouse passar sobre ele.
4. Escreva o CSS para criar um efeito de clique em botões, onde o botão parece ser pressionado para baixo (deslocamento vertical) quando ativo.
5. Crie uma regra CSS que estilize todas as linhas ímpares de uma tabela com fundo cinza claro (efeito zebrado).
6. Desenvolva um código que selecione e estilize apenas o terceiro item de uma lista com fundo amarelo.
7. Escreva o CSS para que todos os campos de input recebam uma borda verde quando estiverem em foco, exceto o botão de envio.
8. Crie uma regra que aplike margem inferior em todos os parágrafos dentro de um artigo, exceto no último parágrafo.
9. Desenvolva um CSS para que os links visitados tenham cor roxa, mas quando em hover voltem a ser azuis.
10. Escreva um código que selecione os primeiros 5 itens de uma lista e aplique negrito neles.

### Slide 211:
1. Crie uma regra CSS que faça os itens pares de uma lista terem um ícone diferente (usando ::before) dos itens ímpares.
2. Desenvolva um CSS para estilizar todos os botões do formulário, exceto o botão "cancelar" que deve ter estilo diferente.
3. Escreva o código que faça um menu dropdown aparecer apenas quando o item principal estiver em hover.
4. Crie uma regra que combine :not() e :nth-child() para estilizar todos os itens de uma lista, exceto o primeiro e o último.
5. Desenvolva um código CSS completo que crie um botão com os seguintes estados:

   * Normal: fundo azul, texto branco
   * Hover: fundo azul escuro, escala 1.05
   * Active: deslocamento para baixo, sombra reduzida
   * Focus: contorno azul claro com outline personalizado
